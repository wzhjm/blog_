hexo+yilia添加RSS订阅功能

2021-12-16

先安装插件

npm install hexo-generator-feed

在hexo根目录配置文件_config.yml中配置插件

# Extensions
plugins:
  hexo-generator-feed
#Feed Atom
feed:
  type: atom
  path: atom.xml
  limit: 20 # 最新的20篇文章

开启主题的左侧边栏的RSS功能

在主题的配置文件theme/yilia/_config.yml中进行设置:

# SubNav
subnav:
  rss: "atom.xml"

重新生成文件

hexo clean, hexo g后会在public目录下生成auto.xml订阅文件，到浏览器进行访问，点击图标会跳转到*/auto.xml界面，显示最近的几篇文章的更新。
